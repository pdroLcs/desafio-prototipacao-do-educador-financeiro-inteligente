# Desafio Prototipação do Educador Financeiro Inteligente

Atue como um especialista em educação financeira digital.

Crie a proposta de um Educador Financeiro Inteligente para jovens adultos e estudantes universitários.

O sistema deve ajudar o usuário a desenvolver organização financeira, economizar dinheiro e aprender a tomar decisões financeiras melhores.

As principais funcionalidades devem incluir:

- Dashboard financeiro com gráficos de gastos
- Planejamento de metas financeiras e controle de orçamento
- Assistente virtual com dicas financeiras personalizadas

A comunicação deve ser leve, moderna, educativa e motivadora.

As respostas devem seguir o formato de explicações curtas, objetivas e acompanhadas de exemplos práticos.

Evite julgamentos financeiros, respostas muito complexas ou recomendações irrealistas.

Ao final, apresente sugestões criativas para transformar essa ideia em um projeto frontend interativo, incluindo:

- Gamificação de economia
- Sistema de conquistas financeiras
- Dashboard com gráficos animados
- Chat inteligente estilo assistente virtual
- Simuladores financeiros interativos
- Interface moderna inspirada em aplicativos bancários digitais
